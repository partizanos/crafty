Getting Started

To get started, you need to follow at least these steps :

  • Install Crafty and the presets you wish to use.
  • Create a crafty.config.js files to declare your JavaScript and CSS bundles.
  • (for Maven projects) Configure your project to use the frontend-maven-plugin to guarantee the version of NPM/Yarn and Node.

Required knowledge

To be able to use Crafty, you need to have a basic understanding of what NPM/Yarn do;

  • Basic knowledge of npm install
  • Know how to run commands installed through NPM (npx your_command or node_modules/.bin/your_command)
  • Understand npm run and the script field
